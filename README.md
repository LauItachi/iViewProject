# iViewProject
iView测试项目

## Stack

* VueJS - 前端搭建
* NodeJS - 应用搭建、JS运行环境
* WebSocket - C/S架构的实现
* MySQL - 数据库操作
* MarkDown - Wiki/文档编写
* WebStorm - JS开发环境
* Chrome for Windows - 网页浏览/调试
